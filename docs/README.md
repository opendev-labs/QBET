## Deployment

The TARS web interface is deployed to Vercel. To deploy your own instance:

1. Click the "Deploy to Vercel" button on the landing page
2. Configure environment variables
3. Deploy and access your TARS instance

## Local Development

To run TARS locally:

```bash
cd ../tars
pnpm install
pnpm dev
```

Visit http://localhost:3000
